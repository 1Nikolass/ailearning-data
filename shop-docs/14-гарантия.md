# Гарантийные условия

## Сроки гарантии по категориям

| Категория | Срок гарантии |
|-----------|--------------|
| Электроинструмент | 2 года |
| Ручной инструмент | 1 год |
| Сантехника (смесители, унитазы) | 3 года |
| Водонагреватели | 2 года |
| Светильники LED | 2 года |
| Электрика (автоматы, розетки) | 1 год |
| Расходники (свёрла, диски, кисти) | Без гарантии |
| Отделочные материалы | Без гарантии |

## Гарантийный случай
Гарантия покрывает заводской брак. НЕ покрывает: повреждения при неправильной эксплуатации, естественный износ, повреждения при самостоятельной установке.

## Как оформить гарантийный случай
1. Клиент присылает фото/видео дефекта
2. Менеджер согласовывает с Лёхой
3. Если подтверждено — замена или возврат денег
4. Срок рассмотрения: до 10 рабочих дней

# Сезонность продаж

## Высокий сезон
- **Апрель–июнь**: ремонтный сезон. Основные продажи: отделочные материалы, инструменты, сантехника.
- **Сентябрь–октябрь**: второй пик ремонтов (до отопительного сезона).

## Низкий сезон
- **Июль–август**: спад (отпуска).
- **Декабрь–февраль**: спад (праздники, холод). Но есть всплеск в январе на «новогодние» скидки.

## Что делать в низкий сезон
- Акции на залежавшийся товар
- Скидки на категории, которые не продаются (зимой — на садовый инструмент)
- Рассылка по базе клиентов с предложениями
- Подготовка к высокому сезону: закупки, обновление каталога

# Как консультировать клиентов

## Общие правила
- Отвечать в течение 1 часа (в рабочее время)
- Быть вежливым, но конкретным
- Если не знаешь ответ — не выдумывай, скажи «уточню и вернусь»
- Всегда предлагать альтернативу, если товара нет

## Частые ситуации

### «Какую дрель выбрать?»
Спроси: для чего? Если для дома (повесить полку, собрать мебель) — шуруповёрт 12V за 3–5 тысяч. Если для ремонта (сверлить бетон) — перфоратор.

### «Какую краску для стен?»
Спроси: какая комната? Для кухни и ванной — моющаяся. Для спальни — обычная интерьерная. Для фасада — фасадная (другой состав).

### «Сколько мне нужно ламината?»
Спроси площадь комнаты. Прибавь 7% на подрезку. Например: комната 20 м² → нужно 21.4 м² → округляем вверх по упаковкам.

### «А можно подешевле?»
Предложи аналог подешевле. Если клиент просит скидку — до 5% можно сразу, от 5% до 10% — спроси Лёху.

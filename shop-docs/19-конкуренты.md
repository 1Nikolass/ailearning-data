# Анализ конкурентов

## Основные конкуренты

### Леруа Мерлен / Лемана ПРО
- Плюсы: огромный ассортимент, низкие цены, узнаваемость
- Минусы: нет персонального подхода, очереди, сложно получить консультацию
- Наше преимущество: персональный менеджер, быстрая доставка, индивидуальные условия

### Петрович
- Плюсы: быстрая доставка, хороший сайт
- Минусы: цены выше среднего
- Наше преимущество: более гибкие цены, скидки для постоянных

### Wildberries / Ozon (маркетплейсы)
- Плюсы: удобство, быстро, всё в одном месте
- Минусы: нельзя проконсультироваться, сложно с возвратами крупногабарита
- Наше преимущество: экспертиза (можем подсказать что купить), простой возврат

## Наша стратегия
Мы не конкурируем по цене с гигантами. Наша сила — **сервис**: быстро ответили, подсказали, доставили, решили проблему. Клиент возвращается не за ценой, а за отношением.
